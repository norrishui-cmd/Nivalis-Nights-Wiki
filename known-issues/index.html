Nivalis Nights Known Issues and Patch Notes Tracker

Known-issues pages can become very useful after release, but they should not speculate before players can test the game. This page is prepared for launch-day support searches and patch note summaries.

Before launch

Record only official warnings, requirements, and developer notes. Do not list rumored bugs.

After launch

Track crashes, save issues, performance problems, quest blockers, controller bugs, and workaround status. Link to official patch notes when available.

Patch summary format

For each update, include date, version if available, key fixes, remaining issues, and whether the fix affects existing saves.
